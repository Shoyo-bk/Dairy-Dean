🔬 Zone 4: Tests for Adequate Pasteurisation

Examine each test — pass or reject! First-try correct = double points!

📖 Park's Quality Standards:
• Phosphatase Test: Must be Negative — clear or colourless = pass ✅; yellow = fail ❌
• Standard Plate Count: 30,000 or fewer organisms per millilitre
• Coliform Count: Zero per 1 millilitre
🎯 First-try correct answer = double points (2×)!
